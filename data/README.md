# Data Folder

Place expression matrices, metadata tables, or embedding matrices here if you want to run the example scripts.

Suggested files:

- `expression_matrix.csv`: rows as samples, columns as genes
- `metadata.csv`: one row per sample, with `sample_id` and covariates
- `embeddings.csv`: rows as texts/documents, columns as embedding dimensions

Do not commit private or sensitive data unless you have permission.
